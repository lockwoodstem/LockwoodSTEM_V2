Units 0–7

Each unit includes a project, certification connection, and major deliverables.

Unit 0

Engineering Design Foundations

Students build the habits, safety practices, documentation routines, and problem-solving mindset used throughout the engineering course.

Project: Aerospace Rescue Challenge
Certification: Engineering Safety Certification

Major deliverables

  • Signed safety agreement
  • Engineering notebook setup
  • Design process reflection
  • Rescue challenge prototype evidence
  • Unit portfolio checkpoint

Unit 1

Visual Communication & Sketching

Students learn to communicate ideas using technical sketching, line conventions, isometric views, orthographic views, and basic dimensions.

Project: Reverse Engineering Challenge
Certification: Technical Sketching Certification

Major deliverables

  • Line convention practice
  • Isometric sketch set
  • Orthographic sketch sheet
  • Reverse engineering drawing
  • Technical sketching certification evidence

Unit 2

Engineering Documentation & Assemblies

Students document parts and assemblies using tolerances, fasteners, adhesives, exploded views, bills of materials, and change records.

Project: Payload Mount Documentation Project
Certification: Engineering Documentation Certification

Major deliverables

  • Tolerance notes
  • Fastener/adhesive selection
  • Exploded assembly documentation
  • BOM
  • Engineering change request
Open Unit 2
Unit 3

Aerospace CAD Modeling

Students use Fusion-style CAD workflows to build accurate part models using sketches, constraints, dimensions, features, patterns, and parameters.

Project: Aerospace Sensor Mount Challenge
Certification: Fusion CAD Level 1

Major deliverables

  • Constrained sketches
  • Feature-based part model
  • Parameterized sensor mount
  • CAD screenshot/export
  • CAD Level 1 certification evidence
Open Unit 3
Unit 4

Technical Drawings & Manufacturing Documentation

Students create manufacturing-ready drawings with projected views, dimensions, notes, section/detail views, and inspection information.

Project: Aerospace Manufacturing Package
Certification: Engineering Drawings Certification

Major deliverables

  • Drawing view layout
  • Dimensioned manufacturing drawing
  • Section/detail view evidence
  • Inspection notes
  • Final drawing package
Open Unit 4
Unit 5

Assemblies & Mechanisms

Students build assemblies and mechanisms using components, joints, motion limits, interference checks, exploded views, and animations.

Project: Aerospace Deployment Mechanism
Certification: Fusion CAD Level 2

Major deliverables

  • Assembly model
  • Joint/motion evidence
  • Interference check
  • Exploded view
  • Mechanism animation
Open Unit 5
Unit 6

Prototyping & Testing

Students prepare, fabricate, test, analyze, and revise prototypes using manufacturing constraints and engineering data.

Project: Aerospace Prototype Validation Challenge
Certification: 3D Printing + Laser Cutter Certifications

Major deliverables

  • Prototype plan
  • Fabrication-ready files
  • Test plan
  • Data table/graph
  • Redesign evidence

Unit 7

Aerospace Systems Design Capstone

Students complete a full engineering project using research, requirements, concept development, CAD, manufacturing documentation, prototyping, testing, revision, and presentation.

Project: Aerospace Launch System Capstone
Certification: Portfolio Completion

Major deliverables

  • Research brief
  • Decision matrix
  • CAD assembly
  • Manufacturing package
  • Test data
  • Final presentation
